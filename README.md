/The raw crime data used in this project is publicly available from:

https://data.police.uk/data/

To reproduce the analysis:

1. Download the street-level crime data from January 2024 – January 2026
2. Select the following police forces:
   - Nottinghamshire Police
   - Leicestershire Police
   - Surrey Police
   - City of London Police
   - Gloucestershire Constabulary

3. Extract the downloaded files and place them inside:

data/raw/

4. Run the preprocessing notebook to generate the cleaned dataset file
